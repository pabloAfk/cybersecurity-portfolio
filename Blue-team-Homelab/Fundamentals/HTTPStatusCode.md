# aqui eu coloco meus conhecimentos em Código de Status HTTP
## HTTP Status Codes são como "notas de atualização" que o servidor envia ao seu navegador pra dizer o que aconteceu com a requisição que você fez, são divididos em cinco categorias principais baseadas na primeira dezena do número

### 1. 1xx
Informativo, significa que a requisição foi recebida e o processo continua

exemplo comum: 101 Switching Protocols

### 2. 2xx
Sucesso, significa que a ação foi recebida, compreendida e aceita com sucesso

exemplo comum: 200 OK

### 3. 3xx
Redirecionamento, significa que o navegador precisa tomar uma ação adicional para completar a requisição

exemplo comum: 301 Moved Permanently

### 4. 4xx
Erro do Cliente, significa que a requisição contém sintaxe incorreta ou não pode ser processada (culpa de quem acessa)

exemplo comum: 404 Not Found

### 5. 5xx
Erro do Servidor, significa que o servidor falhou ao cumprir uma requisição aparentemente válida (culpa do site)

exemplo comum: 500 Internal Server Error
